# 🐍 Snake Game

A responsive Snake Game built using **HTML, CSS, and JavaScript**.

This project dynamically generates the game board using CSS Grid and JavaScript while supporting:
- Responsive resizing
- Food spawning
- Self collision
- Boundary collision
- Pause/Resume system
- Keyboard controls

---

# 🎮 Features

✅ Dynamic responsive grid  
✅ Smooth snake movement  
✅ Random food generation  
✅ Snake growth after eating food  
✅ Collision detection  
✅ Game over system  
✅ Pause / Resume using SPACEBAR  
✅ Responsive board resizing using ResizeObserver  

---

# 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

# 🎹 Controls

| Key | Action |
|---|---|

| ⬇️ Arrow Down | Move Down |

| ➡️ Arrow Right | Move Right |
